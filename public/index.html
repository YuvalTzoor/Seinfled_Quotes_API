Seinfeld Quotes API Documentation

This a brief explanation of how this API works.

Endpoint Example DataType
season/* season/1 Integer
episode/* episode/30 Integer
author/* author/george string
author || season/*/random author/george/random string or integer

Sample Request with request.get method

   // will make a  get request to the quotes endpoint
// and log the response or error
const request = require("request");
request.get("http://localhost:3000/quotes", (err, res, body) => {
	if (err) {
		return console.log(err);
	}
	console.log(JSON.parse(body));
});

Sample Response

    {
    "quote": "Yeah I eat the whole apple.. core, stem, seeds -- everything",
    "author": "Kramer",
